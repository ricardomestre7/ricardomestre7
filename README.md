## Security Guardrails (Always On)
- **No** `--allow-unauthenticated` in production services  
- IAM: **no `allUsers` bindings**; invocation via dedicated roles only  
- CI/CD: **Workload Identity Federation (OIDC)** ‚Äî zero JSON keys in repos  
- Secrets in **Secret Manager**; sensitive env never committed  
- Network isolation; controlled exposure via LB/IAP only when required  
- Separate **staging** and **production** projects with policy boundaries
